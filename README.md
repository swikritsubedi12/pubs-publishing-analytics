# Pubs Publishing Analytics Project

## Project Overview

This project analyzes a publishing company database using SQL, Python, Power BI, and GitHub.

The goal is to understand sales performance across books, publishers, stores, categories, and authors, then provide business recommendations to increase revenue.

## Tools Used

- SQL Server
- SQL Server Management Studio
- Python
- Power BI
- Git
- GitHub

## Current Progress

- Created the `pubs` SQL Server database
- Explored database tables and row counts
- Analyzed revenue, publisher, store, category, and author performance
- Created and validated a reporting view: `vw_sales_detail`

## Main Business Metric

Estimated Revenue = Quantity Sold × Book Price

## Key Tables

- sales
- titles
- publishers
- stores
- authors
- titleauthor
